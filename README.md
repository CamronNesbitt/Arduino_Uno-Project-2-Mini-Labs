# Arduino-Based K-12 Hands-On Learning Modules

Welcome to the Central State University Mini Labs repository! This project features a series of engaging, classroom-tested engineering and programming modules designed to introduce K-12 students to the fundamentals of STEM using the Arduino Uno platform.

---

## 🛠️ Project Deliverables & Highlights

* **K-12 Friendly Curriculum:** Designed and developed engaging, hands-on lesson modules and activities utilizing Arduino Uno kits to teach fundamental engineering and programming concepts.
* **Robust Codebases:** Programmed and stress-tested a variety of baseline application projects—including LED blink patterns, Morse code signaling, and LCD display controls—to ensure classroom reliability and ease of use.
* **Comprehensive Documentation:** Built and maintained this comprehensive GitHub repository featuring well-commented code, step-by-step instructional guides, wiring schematics, and safety documentation.

---

## 📂 Repository Structure

Each "Mini Lab" folder contains the source code (`.ino` files) and its corresponding step-by-step instructional guide (available as a rendered README or PDF format):

* **[Mini Lab 1](./Mini%20Lab%201/): Singular LED Blink**
  * *Concepts:* Digital outputs, delay functions, basic breadboard wiring, and the structure of an Arduino sketch (`setup()` and `loop()`).
* **[Mini Lab 2](./Mini%20Lab%202/): Morse Code Signaling**
  * *Concepts:* Advanced timing delays, patterns, and translating text into tactile/visual signals.
* **[Mini Lab 3](./Mini%20Lab%3/): Push Button Lab**
  * *Concepts:* Digital inputs, pull-up/pull-down resistors, and conditional state logic (`if/else`).
* **[Mini Lab 4](./Mini%20Lab%204/): Multi-LED Sequences**
  * *Concepts:* Managing multiple digital pins, nested loops, and basic sequence programming.
* **[Mini Lab 5](./Mini%20Lab%205/): LCD Display Controls**
  * *Concepts:* Integrating external libraries, parallel data communication, and printing dynamic text to a screen.

---

## 🚦 Safety & Classroom Guidelines

To ensure a seamless and safe learning environment, please refer to the specific safety documentation included in each lab's folder. 

* Always disconnect the Arduino Uno from the USB power source before modifying any physical circuit connections.
* Ensure resistors are correctly placed in series with LEDs to limit current and prevent component damage.
